# aGo Maintenance

WordPress maintenance and coming-soon mode with HTTP 503 + Retry-After. Customizable page with logo, background image, overlay and countdown. **Free, no Pro tier.**

[English](#english) · [Español](#espanol)

---

## English

aGo Maintenance shows a standalone maintenance or coming-soon page to public visitors while admins and editors keep working as normal. Sends HTTP 503 + Retry-After when in maintenance mode (SEO friendly for temporary downtime) or HTTP 200 in coming-soon mode for sites that are not yet launched.

### Features

- Maintenance or coming-soon mode.
- Customizable title, message, colors, logo and background image with darkness overlay.
- Optional countdown timer.
- HTTP 503 + Retry-After (maintenance) or HTTP 200 (coming-soon).
- Admins and logged-in editors always see the live site.
- Cache-busting headers so no full-page cache layer hides the screen.
- No external services.
- English, Spanish (es_ES) and Brazilian Portuguese (pt_BR) bundled.

### Installation

1. Upload `ago-maintenance` to `/wp-content/plugins/`.
2. Activate the plugin.
3. Go to **aGo Tools, Maintenance**. Configure and enable.
